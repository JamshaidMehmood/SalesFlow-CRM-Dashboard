# SalesFlow CRM — Frontend

React + Vite SPA for the SalesFlow CRM Dashboard.

See the [project README](../README.md) for setup, environment variables, features, and demo credentials.

## Local development

```bash
npm install
npm run dev
```

Runs on [http://localhost:5173](http://localhost:5173). Requires the backend API on port 3001.

## Build

```bash
npm run build
npm run preview
```
